NumC


This was a CS61C project that taught me how to use SIMD instructions, loop unrolling, OpenMP, and algorithmic optimization to speed up code. These four charts showcase the possible speed up from different functions. Simple calls like linear math didn't have major speedups, but more complicated calls like power had the potential to be spedup by over 1000x. More specifically, we were supposed to speed up matrix calculations like multiplication and addition. The picture on the right depicts how we should've formatted our matrix in array form.


Gitlet


I implemented the framework to store files with user interaction through methods: add, commit, push, pull, using Java. This was done through the use of unique ID keys and hash values as addresses to store user files that could be pulled from using the pull method.
